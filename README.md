A fully **responsive ticketing website** where users can browse and purchase event tickets online.  

## Features
- **Online store**: Browse and buy tickets directly from the website.
- **Shopping cart & checkout**: Simple, smooth, and user-friendly.
- **Admin panel**: Add, edit, or remove events and users from the database.
- **Responsive design**: Works perfectly on mobile, tablet, and desktop.

## Technologies Used
- HTML & CSS
- Bootstrap
- PHP
- MySQL

## How to Run
1. Clone the repository.
2. Import the database `events_manager.sql` to your MySQL server.
3. Configure `config.php` with your database credentials.
4. Open the website in your browser.
